# HowAboutThisHouse
### 소개
- 프로젝트명 : 이집어때?
- 주제 : 아파트 거래 정보, 주변 상가 정보 등을 조회하는 서비스를 제공하는 부동산 관련 웹 사이트
### 기술스택
    - frontend : vue.js
    - backend : Spring Boot, MySql
    - UI : HTML, CSS
### 주요 기능
- 회원 가입
- 카카오 API를 이용해 실거래가 매물 위치 시각화 
    - 매물 상세 정보 조회
    - 은행, 마트 등 주변 시설 위치 시각화
- 관심 지역 간편 추가/삭제 가능
- 관심 지역 주변 부동산 정보 위치 시각화
- 회원 전용 자유 게시판 기능
    - 댓글과 조회수 기능 제공
- 회원 전용 QnA 게시판 기능
    - 관리자가 일대일로 답변 가능
- 공지사항으로 매물 관련 유용한 정보 제공
### 추후 추가할 기능
- 관심지역 주변 초등학교 정보 공공 API 이용해서 제공
